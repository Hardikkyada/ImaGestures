

# ImaGestures

Using the GestureRecognizer make a very simple app

## Usage

- UITapGestureRecognizer
- UIPinchGestureRecognizer
- UIRotationGestureRecognizer
- UISwipeGestureRecognizer
- UIPanGestureRecognizer
- UIimageview
